# Maneyger

A client-side personal finance tracker for logging **income** and **expenses** and watching
your balance update live — built with vanilla JavaScript, Bootstrap and Font Awesome. No
backend, no build step; everything runs in the browser.

## Features

- Add income and expenditure entries on the fly
- Separate **Income** and **Expenses** views with filters (auto-disabled when a list is empty)
- Live totals as entries change
- Animated intro / logo reveal and a loading spinner
- Responsive Bootstrap 5 layout

## Tech stack

`HTML` · `CSS` · `JavaScript` · `Bootstrap` · `Font Awesome`

## Run locally

It's a fully static site — just open `index.html` in your browser. Or serve the folder:

```bash
npx serve .          # then open the printed URL
```
